# Autonomous CloudOps Copilot using Agentic AI

## Overview
AI-powered incident analysis dashboard for CloudOps.

## Features
- AWS Monitoring
- Kubernetes Pod Monitoring
- Jenkins Status
- Severity Detection
- Root Cause Analysis
- Remediation Suggestions

## Tech Stack
- Python
- Streamlit
- Kubernetes
- AWS
- Agentic AI

## Run Locally

```bash
pip install -r requirements.txt
streamlit run ui/app.py
```
